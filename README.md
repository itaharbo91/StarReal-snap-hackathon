## ğŸª What is a Spectacles Hackathon?

A Spectacles hackathon is a creative challenge where developers and designers collaborate to build innovative **Augmented Reality (AR) experiences** for Snap Spectacles within a limited timeframe.

### The Challenge
Participants create **Lenses** - interactive AR applications that run on Spectacles - using **Lens Studio**, Snap's powerful development platform. These lenses showcase creative use of AR technology to solve real-world problems or create engaging experiences.

### The Tools
- **Lens Studio :** The visual development environment for creating Spectacles lenses
- **Snap SDK :** APIs and modules for advanced features (motion tracking, multiplayer, UI, etc.)
- **Mobile Controller Module :** For precise hand and motion tracking
- **Spectacles Sync Kit :** For multiplayer and collaborative experiences

### The Goal
Transform an idea into a functional, creative AR experience that demonstrates the power of wearable AR technology to enhance how people interact, play, and experience the world.

**StarReal** transforms any venue into an **exclusive sci-fi arena** where each player experiences an epic energy saber duel in augmented reality on Spectacles.
